# VeinMiner 客户端使用说明

> 面向单机/局域网玩家的快速上手指南

## 适用场景

本说明适用于：
- 单人世界
- 局域网联机
- 客户端"对局域网开放"功能

> 如果你使用专用服务器（BDS），请参阅 [README_Server.md](./README_Server.md)

## 快速开始

### 1. 启用连锁
进入世界后，默认连锁已开启。直接潜行 + 用镐挖矿即可触发。

### 2. 打开主菜单
输入：
```
/vein
```

### 3. 切换开关
输入：
```
/vein toggle
```

### 4. 查看状态
输入：
```
/vein status
```

## 个人配置

在客户端模式下，每个玩家拥有完整的个人配置：

### 调整最大连锁数
1. 输入 `/vein`
2. 点击"最大连锁数"
3. 拖动滑块设置（1-128）

### 管理个人白名单
个人白名单 = 共享默认白名单 + 你额外添加的方块

添加方块：
```
/vein whitelist add minecraft:glowstone
```

移除方块：
```
/vein whitelist remove minecraft:glowstone
```

### 重置所有设置
1. 输入 `/vein`
2. 点击"重置设置"

## 触发连锁的条件

必须同时满足：
1. ✅ **潜行状态**：按住 Shift（PC）或双击下蹲（移动端）
2. ✅ **正确工具**：手持镐、斧或锹
3. ✅ **白名单方块**：目标方块在白名单中
4. ✅ **开关开启**：个人开关已开启
5. ✅ **耐久足够**：工具剩余耐久 >= 1

## 支持的方块

### 默认白名单（共享）

**矿石类**：煤矿、铁矿、金矿、钻石矿、绿宝石矿、红石矿、青金石矿、铜矿、下界石英矿、远古残骸（含深层变种）

**原木类**：橡木、云杉木、白桦木、丛林木、金合欢木、深色橡木、红树木、樱花木、绯红菌核、诡异菌核

**石材类**：石头、圆石、深板岩、凝灰岩、玄武岩、黑石、下界岩、末地石、砂岩等

**其他**：玻璃、冰、荧光石、海晶灯、仙人掌、竹子、甘蔗、海带等

## 工具支持

| 工具类型 | 适用方块 |
|----------|----------|
| 镐 | 矿石、石头 |
| 斧 | 原木、木板 |
| 锹 | 泥土、沙子、沙砾 |

## 附魔支持

- **精准采集**：连锁破坏时保留方块本身（如玻璃、冰）
- **时运**：增加矿石掉落数量
- **耐久**：减少工具耐久消耗概率

| 命令 | 说明 |

| `/vein` | 打开主菜单 |
| `/vein toggle` | 快速切换开关 |
| `/vein status` | 查看当前状态 |

## 故障排除

### 问题：连锁不触发
**检查清单**：
- [ ] 是否在潜行？
- [ ] 是否手持镐/斧/锹？
- [ ] 个人开关是否开启？（`/vein status` 查看）
- [ ] 目标方块是否在白名单？

### 问题：连锁数量太少
**解决**：
- `/vein` → 调整最大连锁数
- 检查工具耐久是否足够

### 问题：工具耐久消耗过快
**说明**：
- 每个连锁破坏的方块消耗 1 点耐久
- 使用"耐久"附魔可减少消耗
- 耐久不足时连锁会自动停止

### 问题：游戏卡顿
**解决**：
- 降低最大连锁数
- 关闭连锁，分批挖矿
- 检查电脑性能是否足够

### 问题：行为包无法加载
**检查**：
- 游戏版本是否 >= 1.21.100
- 是否启用了"测试版 API"
- 行为包是否在 `development_behavior_packs/` 目录

### 问题：客户端模式下显示"服务端模式"命令不可用
**说明**：这是正常现象。客户端模式下，以下命令不可用：
- `/vein blacklist`
- `/vein whitelist`（额外白名单）
- `/vein limit`
- `/vein rate`
